# How you feel? Sweet Parameter massage.

This program take an input of how you feel today. and it'll give you a lovely message for running your day tommorrow.

## Usage
```js
import { SweetMessage } from "sweetMessage";

window.onload = () => {
    SweetMessage();
}
```
